# MusicX
🎵 MusicX: 多模态音乐理解与疗愈生成系统

MusicX 是一个基于 NVIDIA AI 技术构建的多模态音乐智能系统，融合音乐理解、生成与情绪调节能力，致力于打造面向数字疗愈场景的全新交互范式。系统支持音乐分析、音乐对话、个性化音乐生成以及音画融合展示，通过“理解—生成—反馈”的闭环机制，使人工智能从内容生成工具演进为可参与情绪调节的智能系统。

本项目基于 NVIDIA NeMo 框架 构建，集成 Music Flamingo（音乐理解）、TangoFlux（音乐生成） 与 NemoTron-3 Nano（大语言模型），并通过多Agent协同机制实现统一调度。同时，系统结合疗愈音乐数据进行模型微调，并引入视觉生成与沉浸式交互模块，实现跨模态体验增强。

在工程实现上，前端基于 Vue3 构建，支持复杂多模态交互与实时反馈；后端基于 Django，负责模型调度、异步任务处理与多模型协同推理。系统整体具备良好的扩展性与可部署性，可应用于音乐教育、内容创作与心理健康等多个场景。
